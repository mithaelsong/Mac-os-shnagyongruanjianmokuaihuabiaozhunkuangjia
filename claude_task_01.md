### 1. 主程序启动 (01_主程序启动.swift)
- XRZApplication 单例类

### 2. 日志系统 (02_初始化日志系统.swift)
- LogSystem 单例
- 支持多级日志（debug/info/warning/error/fatal）
- 控制台输出 + 文件输出（7天自动轮转）
- 异步写入（不阻塞主线程）

### 3. 配置系统 (03_初始化配置系统.swift)
- ConfigSystem 单例
- 支持 ModuleConfig（启用/禁用、优先级、依赖、自定义设置）
- 从 Info.plist 和 UserDefaults 读取
- 持久化到 JSON 文件

### 4. 扫描模块目录 (04_扫描模块目录.swift)
- ModuleScanner 类
- 扫描 PlugIns/ 目录下所有模块文件夹
- 读取 ModuleMetadata.json
- 验证 bundle 文件存在
- 按优先级排序返回

- ModuleLoader 类
- 先加载核心模块（优先级 < 50），再加载业务模块（优先级 >= 50）
- 检查依赖是否已加载
- 调用模块 start() 方法
- 失败不崩溃，继续加载其他
- 记录加载时间
